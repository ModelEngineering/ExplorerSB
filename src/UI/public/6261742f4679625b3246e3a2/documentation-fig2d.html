This is the general mathematical model for the AE1 model with modifications to enable simulations which are capable of reproducing Figure 2D in the original paper. See the various views available in the right hand column. The simulation experiment for this model can be obtained by loading the corresponding SED-ML document into OpenCOR and executing the simulation. The results of which are shown below. A rendering of the mathematical model itself is available here.

In Figure 2D, the HCO3- influx is computed for a range of internal HCO3- when ambient Cl- is not present. External HCO3- concentration is fixed at 50 mM.
